# Quickstart

This repository now includes the minimum Mintlify project structure required for setup.

## Recommended first pass

1. Define the top-level docs sections you actually need.
2. Write one page for onboarding, one for concepts, and one for integration details.
3. Keep the content product-focused and agent-first from the start.

## Suggested structure

- Getting started
- Concepts
- API or integrations
- Use cases
- FAQ
